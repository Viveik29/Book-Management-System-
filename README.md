# Book-Management-System-
Admin +User modules 
## 📌 Overview
This is a command-line based Book Management System built using Python.
It allows Admin and Users to manage books efficiently using CSV file storage.

## 🚀 Features

### Admin:
- Add books
- Update book details
- Delete books

### User:
- View available books
- Borrow books
- Return books

## 🗂️ Project Structure
- admin.py
- user.py
- file_handler.py
- main.py
- library_dataset_updated.csv

## ▶️ How to Run

```bash
python main.py
